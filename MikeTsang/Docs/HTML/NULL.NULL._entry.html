_entry Function

This is the main entry point. After the processor is reset, it jumps to the code located at RESET_VECTOR which is where this code is located. This start up location contains only one instruction. It immediately jumps to the _startup function which is located in the _startup code section.

Syntax

void _entry(
	void  //param element not supplied
);

Return Value

None.
