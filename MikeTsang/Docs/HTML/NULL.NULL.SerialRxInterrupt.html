SerialRxInterrupt Function

This function enables or disable receive interrupt for the serial port.

Syntax

void SerialRxInterrupt(
	BYTE bSerialPort,
	BYTE fEnable
);

Parameters

bSerialPort
Specifies the serial port.
fEnable
If 1, enable interrupt. If 0, disable interrupt.

Return Value

None.
