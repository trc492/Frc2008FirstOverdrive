_MoveTo

This structure contains the MoveTo state machine info.

Fields

bState
Stores the state of the state machine.
lTargetHeading
Stores the target heading.
lTargetPos
Stores the target position.
iTurnVel
Stores the turning velocity.
iDriveVel
Stores the driving velocity.
iEndVel
Stores the ending velocity.
iTargetVel
Stores the ending velocity.
lTurnThreshold
Stores the current target velocity for PID calculation.
lAngleTolerance
Stores the angle tolerance for which it will be considered heading on target if we don't exceed it.
lEndZonePos
Stores the end zone position where we need to change the velocity to EndVel allowing the distance to accelerate or decelerate.
