_user_putc Function

This function overrides the putc() interface to C18 2.4 output stream functions and writes a byte to the serial port's transmit queue.

Syntax

void _user_putc(
	BYTE bData
);

Parameters

bData
Specifies the data to write to the serial port.

Return Value

None.
