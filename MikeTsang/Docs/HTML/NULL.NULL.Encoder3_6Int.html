Encoder3_6Int Function

This function handles the encoder 3 interrupt. If enabled, it is called when the interrupt pin changes logical state.

Syntax

void Encoder3_6Int(
	BYTE bChannel,
	BYTE bPhaseBPin,
	BYTE bState
);

Parameters

bChannel
Specifies the encoder channel to get its cound.
bPhaseBPin
Specifies the state of the phase B pin.
bState
Specifies the phase A pin state.

Return Value

None.
