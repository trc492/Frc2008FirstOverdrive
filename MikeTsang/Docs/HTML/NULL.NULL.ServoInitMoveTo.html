ServoInitMoveTo Function

This function initializes the MOVETO structure for autonomous drive.

Syntax

void ServoInitMoveTo(
	PMOVETO MoveTo,
	PDRIVE Drive,
	long lHeading,
	long lDistance,
	int iTurnVel,
	int iDriveVel,
	int iEndVel,
	long lTurnThreshold,
	long lAngleTolerance
);

Parameters

MoveTo
Points to the MOVETO structure to be initialized.
Drive
Points to the DRIVE structure that contains the motor information.
lHeading
Specifies the absolute heading we should be at after the state machine ended.
lDistance
Specifies the absolute distance we should be at after the state machine ended.
iTurnVel
Specifies the turning velocity.
iDriveVel
Specifies the velocity when driving in a straight line.
iEndVel
Specifies the ending velocity when the state machine ended.
lTurnThreshold
Specifies the amount of turn we need to exceed before we classify it being a turn. This is to determine if we need to slow down to turn velocity. If we are just adjusting our heading when driving a straight line, that's not turning and we don't need to slow down.
lAngleTolerance
Specifies the amount of angle tolerance we allow comparing to the target heading. This is to prevent the oscillation caused by minor angle variation.

Return Value

None.
