ADCInitialize Function

This function initializes the ADC hardware.

Syntax

void ADCInitialize(
	WORD wSampleRate,
	WORD wSamplesPerUpdate,
	WORD wChannelMask
);

Parameters

wSampleRate
Specifies the sample rates of all ADC channels.
wSamplesPerUpdate
Specifies the number of samples per update (oversampling rate). Must be a power of 2 from 1 to 256.
wChannelMask
Specifies the bit map of in-use ADC channels.

Return Value

None.

Remarks

Number of ADC samples that will be averaged for each update. More will generally give you more resolution and less noise (see chart below), but your update rate will decrease proportionately.
