SerialTxIntHandler Function

This function is called when there is a transmit empty interrupt on the serial port.

Syntax

void SerialTxIntHandler(
	BYTE bSerialPort
);

Parameters

bSerialPort
Specifies the serial port to get the byte count.

Return Value

None.

Remarks

If the interrupt handler was installed correctly, this function will be called every time the serial port is ready to start sending a byte of data.
