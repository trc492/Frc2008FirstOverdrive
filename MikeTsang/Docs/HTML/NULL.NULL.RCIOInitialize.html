RCIOInitialize Function

This function configures the RC I/O such as digital I/O and relays.

Syntax

void RCIOInitialize(
	DWORD dwOutChannelMask
);

Parameters

dwOutChannelMask
Specifies the bit mask of the digital out channels. Note: The caller should OR the RC_DIGITAL_* values for this parameter.

Return Value

None.
