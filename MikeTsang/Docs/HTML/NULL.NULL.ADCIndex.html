ADCIndex Function

This function finds the ADC channel from the ADC array and returns the index to it.

Syntax

int ADCIndex(
	BYTE bChannel
);

Parameters

bChannel
Specifies the ADC channel to look for.

Return Value

Success: Returns the channel index.

Return Value

Failure: Returns -1.
