Serial2Initialize Function

This function initializes serial port two.

Syntax

void Serial2Initialize(
	BYTE bBaudRate,
	BYTE bFlags
);

Parameters

bBaudRate
Specifies the baud rate to initialize the serial port.
bFlags
Specifies the option flags for initializing the serial port.

Return Value

None.

Remarks

This function must be called before you try to use serial port two. By default, this serial port is set to 9600 baud with the transmitter and receiver enabled. The serial port's baud rate is programmed by entering a value into the SPBRG2 register and possibly changing the value of the BRGH bit. Several example values are included in the serial_ports.h file. Numbers within brackets refer to the PIC18F8722 data sheet page number where more information can be found. This document can be found at microchip's website at http://www.microchip.com or at the author's website at http://www.kevin.org/frc.
