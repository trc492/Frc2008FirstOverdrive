CopyPgmString Function

This function copies a string from program memory to data memory. String literals are allocated in program memory. A lot of library functions can only access data memory.

Syntax

PSTR CopyPgmString(
	PSTR pszStr,
	BYTE bLen,
	const rom char * pszPgmStr
);

Parameters

pszStr
Points to string buffer in data memory.
bLen
Specifies the size of the string buffer.
pszPgmStr
Points to string in program memory.

Return Value

Returns a copy of the string in data memory.
