RCIOPrintRelay Function

This function prints the state of the RC relay.

Syntax

void RCIOPrintRelay(
	BYTE bRelayChannel,
	BYTE bFwd,
	BYTE bRev
);

Parameters

bRelayChannel
Specifies the relay channel.
bFwd
Specifies the Fwd state of the relay.
bRev
Specifies the Rev state of the relay.

Return Value

None.
