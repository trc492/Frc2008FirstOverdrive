_EncChannel

This structure stores the states of an encoder channel.

Fields

bChannel
Stores the encoder channel number (0-based).
bState
Stores the phase A pin state. This field is only used for encoder channels 3-6.
fReverse
Specifies if the encoder direction is reversed.
lPrevCount
Stores the previous encoder count. This field is used for calculating the velocity by calculating the difference between the current and previous encoder counts.
lCount
Stores the current encoder count.
lVelocity
Stores the calculated velocity for the current system timer tick.
