GyroGetRate Function

This function returns the current angular rate of change of the given gyro in unit of 0.1 deg/sec or milliradians/sec depending on the gyro sensitivity unit passed to the GyroInitialize function.

Syntax

int GyroGetRate(
	void  //param element not supplied
);

Return Value

Returns the gyro rate.
