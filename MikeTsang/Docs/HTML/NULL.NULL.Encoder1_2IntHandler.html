Encoder1_2IntHandler Function

This function handles the encoder 1 interrupt. If enabled, it is called when the interrupt pin changes from a logic 0 to a logic 1.

Syntax

void Encoder1_2IntHandler(
	BYTE bChannel,
	BYTE bPhaseBPin
);

Parameters

bChannel
Specifies the encoder channel to get its cound.
bPhaseBPin
Specifies the state of the phase B pin.

Return Value

None.
