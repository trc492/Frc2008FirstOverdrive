ServoPidDrive Function

This macro applies the PID algorithm to calculate the new drive value and normalize it to the PWM value and adjust the motor direction.

Syntax

ServoPidDrive(
	macro_parameter Servo,
	macro_parameter lPos,
	macro_parameter lVel,
	macro_parameter fPosPid
);

Parameters

Servo
Points to the SERVO structure.
lPos
Specifies the current position. This is ignore if fPosPid is FALSE.
lVel
Specifies the current velocity.
fPosPid
If TRUE, apply position PID, otherwise apply velocity PID.

Return Value

Returns the new drive value.
