GyroCalibrate Function

This function initialize the gyro for calibration.

Syntax

void GyroCalibrate(
	void  //param element not supplied
);

Return Value

None.

Remarks

IMPORTANT: Only call this function when your robot will be absolutely still and free of vibration (e.g. the air compressor is off) until the call to GyroStopCal is made.
