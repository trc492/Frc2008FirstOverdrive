InitializePort Function

This function initializes the joystick port and prepare for calibration. It specifies which analog channels of the joystick port are in use.

Syntax

void InitializePort(
	BYTE bPort,
	BYTE bPortFlags
);

Parameters

bPort
Specifies the joystick port for calibration. Note: port number is 0-based.
bPortFlags
Specifies which analog channels of the joystick port should be calibrated.

Return Value

None.
