InputInitialize Function

This function initializes the input data. This includes calibrating all in-use analog channels. The Operator Interface has 4 joystick ports and each has 4 analog channels totalling 16 analog channels. This function allows specifying which joystick ports and which analog channels are in used so that only those channels are initialized and calibrated.

Syntax

void InputInitialize(
	BYTE bPort1Flags,
	BYTE bPort2Flags,
	BYTE bPort3Flags,
	BYTE bPort4Flags
);

Parameters

bPort1Flags
Specifies which analog channels of joystick 1 are in use.
bPort2Flags
Specifies which analog channels of joystick 2 are in use.
bPort3Flags
Specifies which analog channels of joystick 3 are in use.
bPort4Flags
Specifies which analog channels of joystick 4 are in use.

Return Value

None.
