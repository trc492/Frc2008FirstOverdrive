SerialRxIntHandler Function

This function is called when there is a receive interrupt on the serial port.

Syntax

void SerialRxIntHandler(
	BYTE bSerialPort
);

Parameters

bSerialPort
Specifies the serial port to get the byte count.

Return Value

None.

Remarks

If the interrupt handler was installed correctly, this function will be called every time a new byte of data is received by serial port.
